Авторизация
/api/auth
/register  -  POST  - Принимает обязательные параметры: login, email, password, teacher: String, role: Boolean
Регистрирует пользователя в бд и отправляет на его почту сообщение

/login  -  POST - Принимает обязательные параметры: login, password: String
Возвращает токен авторизации. В последующем посылать его в хедерах для авторизации пользователя
Authorization: Bearer token
const headers = {authorization: `Bearer ${token}`}
